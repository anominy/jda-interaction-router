# JDA Interaction Router

## Install
### Maven
```xml
<dependency>
    <groupId>io.github.anominy</groupId>
    <artifactId>jda-interaction-router</artifactId>
    <version>0.1-r0</version>
</dependency>
```
### Gradle
```groovy
implementation 'io.github.anominy:jda-interaction-router:0.1-r0'
```
